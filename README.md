# python-trojan
При запуске данный вирус запрашивает права администратора, после чего начинает работать в фоне.

Данный троян удаляет все файлы с компьютера которые может, после убивает процесс Taskmgr.exe, explorer.exe и открывает полноэкранное красное окно с надписью
"Ваш пк был заражён вирусом и уничтожен, перезагрузка = смерть.
Все ваши файлы были удалены, некоторые системные файлы были повреждены, из-за чего компьютер больше не запустится
Попытка закрыть окно приведёт к перезагрузке компьютера
Recovery не сможет восстановить пк"

окно работает повверх всех программ

Блокируются клавиши:
win (windows)
ctrl
shift
alt
esc
tab
del (delete)

При нажатии на заблокированную клавишу, компьютер перезагрузится. Из-за того, что некоторые системные Файлы были удалены, компьютер больше не запустится, а Recovery не починит это
